# 動画配信ファミリープラン比較｜家族で使うならどれがお得？

動画配信サービスって、一人で使うのと家族で使うのとでは、選び方が全然違うんだよね。

一人ならコンテンツの好みだけで選べばいいけど、家族だとそうはいかない。お父さんは海外ドラマが見たい、お母さんは韓流ドラマ、子供はアニメ、おじいちゃんは時代劇……全員の希望を叶えるサービスはあるの？

答えは**ある**。ただし、**1つのサービスで全員満足は難しい**から、家族構成と視聴スタイルに合った選び方が大事。

この記事では、主要な動画配信サービスの「家族向け機能」を徹底比較する。同時視聴数、プロフィール機能、キッズ設定、1人あたりのコスパまで全部チェックしていくよ。

## 家族で使う時に重要な4つのポイント

### ポイント1: 同時視聴数

家族でテレビ、スマホ、タブレットなど別々のデバイスで同時に見られるかどうか。これが少ないと「お父さんが見てるから見られない」問題が発生する。

### ポイント2: プロフィール数

1つのアカウントで複数のプロフィールを作れるかどうか。プロフィールごとに視聴履歴とおすすめが分かれるから、家族それぞれが自分好みのコンテンツを見つけやすくなる。

### ポイント3: キッズモード

子供が不適切なコンテンツを見ないように制限できる機能。子供がいる家庭では必須。

### ポイント4: 1人あたりのコスパ

月額料金を家族の人数で割った「1人あたりコスト」。これで比較すると、高く見えるサービスが実は最安だったりする。

## 各サービスのファミリー機能比較

### Netflix

| 項目 | 内容 |
|------|------|
| 月額料金 | 広告つき 890円 / スタンダード 1,590円 / プレミアム 2,290円 |
| 同時視聴数 | 広告つき 2台 / スタンダード 2台 / プレミアム 4台 |
| プロフィール数 | 最大5つ |
| キッズプロフィール | あり（年齢制限設定可能） |
| ダウンロード | 対応（端末数制限あり） |
| 1人あたり（4人家族） | プレミアムで約573円 |

**家族向けの評価**: プレミアムプランなら4台同時視聴可能で、プロフィールも5つ作れる。キッズプロフィールの制限もしっかりしている。ただし、月2,290円はやや高い。

### U-NEXT

| 項目 | 内容 |
|------|------|
| 月額料金 | 2,189円（毎月1,200ポイント付与） |
| 同時視聴数 | 4台（ファミリーアカウント） |
| プロフィール数 | 親アカウント1 + 子アカウント3 = 最大4つ |
| キッズプロフィール | あり（購入制限、視聴制限） |
| ダウンロード | 対応 |
| 1人あたり（4人家族） | 約547円（実質: ポイント差引で約247円） |

**家族向けの評価**: **家族利用のコスパは全サービス中最強**。4アカウント同時視聴で、子アカウントごとに購入制限・視聴制限をかけられる。毎月1,200ポイントを差し引くと1人あたり実質247円。圧倒的。

### Amazon Prime Video

| 項目 | 内容 |
|------|------|
| 月額料金 | 600円（年払い5,900円/月約492円） |
| 同時視聴数 | 3台 |
| プロフィール数 | 最大6つ |
| キッズプロフィール | あり |
| ダウンロード | 対応 |
| 1人あたり（4人家族） | 約150円（年払い: 約123円） |

**家族向けの評価**: 1人あたりのコストが激安。プロフィールも6つ作れて柔軟。ただし、コンテンツ量は専業サービスに比べると見劣りする面も。

### Disney+

| 項目 | 内容 |
|------|------|
| 月額料金 | スタンダード 990円 / プレミアム 1,320円 |
| 同時視聴数 | スタンダード 2台 / プレミアム 4台 |
| プロフィール数 | 最大7つ |
| キッズプロフィール | あり |
| ダウンロード | 対応 |
| 1人あたり（4人家族） | プレミアムで約330円 |

**家族向けの評価**: ディズニー、ピクサー、マーベルの作品は子供から大人まで楽しめるから、家族の共有コンテンツとして最適。プロフィール数も7つと多い。

### Hulu

| 項目 | 内容 |
|------|------|
| 月額料金 | 1,026円 |
| 同時視聴数 | 4台 |
| プロフィール数 | 最大6つ |
| キッズプロフィール | あり |
| ダウンロード | 対応 |
| 1人あたり（4人家族） | 約257円 |

**家族向けの評価**: 4台同時視聴で月1,026円は優秀。国内ドラマが充実しているから、家族でリアルタイムのドラマを追いかけやすい。

## キッズ機能の比較

子供がいる家庭では、キッズ機能の充実度が重要。

### Netflix のキッズ機能

- **キッズプロフィール**: 年齢に応じたコンテンツのみ表示
- **PINロック**: プロフィール切り替えにPINコードが必要（大人のプロフィールを子供が使えないように）
- **視聴制限**: 年齢レーティングに基づくフィルタリング
- **視聴時間制限**: なし（親が管理する必要あり）

**評価**: キッズ向けの仕組みは整っている。ただし視聴時間の制限機能はない。

### U-NEXT のキッズ機能

- **子アカウント**: R18/R20コンテンツの視聴制限
- **購入制限**: 子アカウントではポイント購入不可に設定可能
- **セキュリティコード**: アカウント切り替えにコードが必要

**評価**: 特にアダルトコンテンツの制限がしっかりしているのが安心。子アカウントからはアクセスできない仕組み。

### Amazon Prime Video のキッズ機能

- **Amazon Kidsプロフィール**: 完全に子供向けのUIに切り替わる
- **年齢制限**: 設定した年齢に応じたコンテンツのみ表示
- **視聴時間制限**: 設定可能
- **PIN保護**: プロフィール切り替えにPINが必要

**評価**: **視聴時間制限ができるのはAmazon Prime Videoだけ**（主要サービスの中で）。子供の見すぎを防ぎたい親にはありがたい機能。

### Disney+ のキッズ機能

- **キッズプロフィール**: 子供向けコンテンツのみ表示
- **プロフィールPIN**: 設定可能
- **コンテンツの安全性**: そもそもディズニーのコンテンツは家族で安心して見られるものが多い

**評価**: ディズニーブランドの安心感が最大の強み。コンテンツ自体が家族向けだから、フィルタリングの心配が少ない。

## よくある質問

### Q. アカウント共有は規約違反？

同居家族での共有は、ほとんどのサービスで認められている。ただし、**別居の友人や家族とのアカウント共有は規約違反になるサービスが多い**。Netflixは特にアカウント共有の制限を強化しているので注意。

### Q. ダウンロード機能は家族全員使える？

ほとんどのサービスで、各プロフィールごとにダウンロードが可能。ただし、端末数やダウンロード本数に上限がある場合がある。

### Q. 子供が勝手に有料コンテンツを購入しないか心配

U-NEXTの子アカウントは購入制限が設定可能。Amazon Prime Videoも購入時にPINを要求する設定ができる。必ず設定しておこう。

### Q. テレビの取り合いになる場合は？

Fire TV Stickを複数台購入するのがコスパ最強の解決策。5,000〜10,000円の追加投資で、各部屋で別々のコンテンツを楽しめるようになる。
